# 1. SEO Overview

The SEO strategy of Darik Pars is designed to increase organic visibility, establish financial authority, attract qualified investors, and build long-term search engine trust.

The SEO strategy focuses on:

- Technical excellence
- High-quality financial content
- Investor education
- Search intent optimization
- Brand authority


Main objective:

"Become a trusted digital destination for investment knowledge and asset management solutions."


---

# 2. SEO Goals


## Brand Visibility

Increase awareness of Darik Pars in financial search results.


## Organic Traffic Growth

Generate qualified visitors through search engines.


## Investor Acquisition

Convert organic visitors into consultation requests.


## Authority Building

Create trust through expert financial content.

# 3. Technical SEO Architecture


# Website Structure


Requirements:

- Clean URLs
- Logical hierarchy
- Fast loading pages
- Mobile optimization
- Crawlable architecture


Example:


Good:

/funds/fixed-income-fund


Bad:

/page?id=123



---

# 4. URL Strategy


Rules:


- Use English slugs
- Keep URLs short
- Avoid unnecessary parameters
- Use keyword-focused paths



Examples:


Company:

/about


Services:

/services/portfolio-management


Funds:

/funds/fund-name


Articles:

/insights/article-title



---

# 5. Metadata Strategy


Every page must include:


## SEO Title

Structure:


Primary Keyword + Brand Name


Example:

---

## Meta Description


Requirements:


- 140-160 characters
- Clear value proposition
- Include target keyword
- Include CTA when possible
